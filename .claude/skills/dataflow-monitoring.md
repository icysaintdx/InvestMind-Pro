# 数据流监控优化 Skill

## 概述
本 skill 用于优化 InvestMindPro 的数据流监控模块，实现高时效性的新闻、公告监控和智能通知。

## 核心功能

### 1. 刷新频率设置
用户可以自定义以下刷新频率：
- **新闻监控**: 30秒 / 1分钟 / 2分钟 / 5分钟
- **公告监控**: 1分钟 / 2分钟 / 5分钟 / 10分钟
- **其他数据**: 5分钟 / 10分钟 / 30分钟 / 1小时

### 2. 静默刷新
自动刷新时不显示加载遮罩，用户无感知。

### 3. 数据源优先级
1. 财联社快讯 - 最快的财经快讯
2. 东方财富新闻 - 覆盖面广
3. 巨潮资讯公告 - 权威来源
4. 新浪/同花顺/富途 - 补充数据

## 使用方法

### 调整刷新频率
1. 点击数据流页面右上角的"⏱️ 刷新设置"按钮
2. 选择各类数据的刷新频率
3. 点击"保存设置"

### 建议配置
- **盯盘时**: 新闻30秒，公告1分钟
- **日常监控**: 新闻1分钟，公告2分钟
- **后台运行**: 新闻5分钟，公告10分钟

## 技术实现

### 前端
- 使用 `setInterval` 实现定时刷新
- 设置保存到 `localStorage`
- 组件卸载时清理定时器

### 后端
- 新闻数据5分钟缓存
- 支持 SSE 流式传输
- 多数据源并行获取

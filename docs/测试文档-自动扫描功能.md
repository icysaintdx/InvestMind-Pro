## 功能说明

这是一个测试文档，用于验证文档自动扫描功能是否正常工作。

### 自动扫描特性

1. **实时扫描**: 后端API自动扫描 `docs/` 目录
2. **自动分类**: 
   - 匹配预定义模式的文档归入对应分类
   - 未匹配的文档归入"其他文档"分类
3. **动态更新**: 新增文档后刷新页面即可看到

### 测试步骤

1. 在 `docs/` 目录创建新的 `.md` 文件
2. 刷新文档中心页面
3. 新文档应该出现在"其他文档"分类中
4. 点击即可查看内容

## 技术实现

### 后端API
```python
@router.get("/list")
async def list_documents():
    # 扫描docs目录
    md_files = []
    for file in DOCS_DIR.glob("*.md"):
        if file.is_file():
            md_files.append(file.name)
    return {"files": md_files}
```

### 前端处理
```javascript
// 1. 获取所有文件
const allFiles = response.data.files

// 2. 按规则分类
const categorizedDocs = categoryConfig.map(cat => ({
  docs: allFiles.filter(file => cat.pattern.test(file))
}))

// 3. 收集未分类文档
const uncategorizedDocs = allFiles.filter(file => !categorizedFiles.has(file))

// 4. 添加"其他文档"分类
if (uncategorizedDocs.length > 0) {
  categorizedDocs.push({
    name: '其他文档',
    docs: uncategorizedDocs
  })
}
```

# 前端文档功能说明

> **创建日期**: 2025-12-05  
> **功能**: 在线文档查看系统

---

## 功能概述

在前端页面添加了**文档中心**功能，用户可以直接在线查看所有项目文档，无需离开应用。

---

## 使用方法

### 1. 打开文档中心

在页面顶部导航栏，Logo右侧有两个按钮：
- **ℹ️ 项目介绍按钮**（紫色）
- **📚 文档中心按钮**（绿色）← 新增

点击 **📚 文档中心按钮** 即可打开文档查看器。

### 2. 浏览文档

文档中心采用**左右分栏布局**：

#### 左侧：文档目录
- 按类别组织（智能体提示词、核心算法、项目介绍）
- 点击文档名称即可查看
- 当前选中的文档会高亮显示

#### 右侧：文档内容
- Markdown格式渲染
- 支持代码高亮
- 支持表格显示
- 自动滚动

## 技术实现

### 前端组件
- **DocumentView.vue**: 文档查看器主组件
- **documents.js**: 文档数据配置

### 核心功能
1. **Markdown渲染**: 使用 `marked` 库
2. **左右分栏**: CSS Grid布局
3. **响应式设计**: 自适应不同屏幕
4. **代码高亮**: 内置样式支持
5. **表格美化**: 自定义表格样式

### 样式特点
- **深色主题**: 与系统整体风格一致
- **渐变效果**: 绿色主题（区别于项目介绍的紫色）
- **平滑动画**: 悬停和选中效果
- **滚动条美化**: 自定义滚动条样式

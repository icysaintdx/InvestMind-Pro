## 🎯 数据源分类

### 1. 法律诉讼风险 ⚖️

#### 1.1 中国裁判文书网 ⭐⭐⭐⭐⭐
- **网址**: https://wenshu.court.gov.cn/
- **数据类型**: 
  - 民事案件
  - 刑事案件
  - 行政案件
  - 执行案件
- **费用**: 免费
- **难度**: 高（需要复杂加密）
- **更新频率**: 实时
- **数据量**: 超过1亿份文书
- **状态**: 框架已完成，待实现加密
- **API**: 需要逆向工程

#### 1.2 AKShare 裁判文书 ⭐⭐⭐⭐
```python
import akshare as ak
# 获取裁判文书数据
df = ak.stock_zh_a_cpws(symbol="股票代码")
```
- **费用**: 免费
- **难度**: 低（已封装）
- **限制**: 数据可能不完整
- **状态**: 可直接使用

## 🛠️ 技术实现方案

### 方案A：直接爬取（推荐）
**适用**: 中国执行信息公开网、企业信用公示系统、证监会

**优势**:
- 数据最新
- 完全免费
- 可控性强

**劣势**:
- 需要处理反爬虫
- 维护成本高

### 方案B：使用AKShare（快速）
**适用**: 失信被执行人、被执行人

**优势**:
- 开箱即用
- 维护成本低
- 稳定性好

**劣势**:
- 数据可能不完整
- 更新频率不可控

### 方案C：混合方案（最优）
- 核心数据：直接爬取
- 补充数据：使用AKShare
- 备用方案：API调用

## 💡 使用建议

### 风险评分模型
```python
风险总分 = (
    失信被执行人 * 0.3 +
    证监会处罚 * 0.25 +
    重大诉讼 * 0.2 +
    经营异常 * 0.15 +
    其他风险 * 0.1
)
```

### 风险等级划分
- **极高风险** (80-100分): 失信被执行人 + 证监会处罚
- **高风险** (60-80分): 重大诉讼 + 经营异常
- **中风险** (40-60分): 一般诉讼 + 行政处罚
- **低风险** (0-40分): 无重大风险事项

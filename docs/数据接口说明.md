## 实时盘口TICK快照(爬虫版)
实时盘口TICK快照(爬虫版)
接口：realtime_quote，A股实时行情
描述：本接口是tushare org版实时接口的顺延，数据来自网络，且不进入tushare服务器，属于爬虫接口，请将tushare升级到1.3.3版本以上。
权限：0积分完全开放，但需要有tushare账号，如果没有账号请先注册。
说明：由于该接口是纯爬虫程序，跟tushare服务器无关，因此tushare不对数据内容和质量负责。数据主要用于研究和学习使用，如做商业目的，请自行解决合规问题。



输入参数

名称	类型	必选	描述

## 实时成交数据(爬虫版)
实时成交数据(爬虫版)
接口：realtime_tick，由于采集和拼接当日以来的成交全历史，因此接口提取数据时需要一定时间，请耐心等待。请将tushare升级到1.3.3版本以上。
描述：本接口是tushare org版实时接口的顺延，数据来自网络，且不进入tushare服务器，属于爬虫接口，数据包括该股票当日开盘以来的所有分笔成交数据。
权限：0积分完全开放，但需要有tushare账号，如果没有账号请先注册。
说明：由于该接口是纯爬虫程序，跟tushare服务器无关，因此tushare不对数据内容和质量负责。数据主要用于研究和学习使用，如做商业目的，请自行解决合规问题。



输入参数

名称	类型	必选	描述
ts_code	str	N	股票代码，需按tushare股票代码标准输入，比如：000001.SZ表示平安银行，600000.SH表示浦发银行，单次只能输入一个股票
src	str	N	数据源 （sina-新浪 dc-东方财富，默认sina）


输出参数

名称	类型	描述
time	str	交易时间
price	float	现价
change	float	价格变动
volume	int	成交量（单位：手）
amount	int	成交金额（元）
type	str	类型：买入/卖出/中性

## 停复牌信息：获取个股每日停复牌状态，直接反映公司是否因重大风险（如违规、经营问题）导致交易暂停，是核心风险信号。
每日停复牌信息
接口：suspend_d
更新时间：不定期
描述：按日期方式获取股票每日停复牌信息

end_date	str	N	停复牌查询结束日期

suspend_timing	str	Y	日内停牌时间段
suspend_type	str	Y	停复牌类型：S-停牌，R-复牌

# 一、新闻舆情类接口（直接获取个股相关资讯）
接口名称	接口标识	核心功能	风险 / 新闻关联价值
## 上市公司公告大全	stock_announcement	获取沪深 A 股上市公司最新公告（含业绩预告、重大事项、违规提示等）	直接抓取公司官方披露的风险事件（如违规公告、诉讼公告）、重要动态，是个股风险第一手来源


## 个股新闻资讯	stock_news_em（东方财富）/ stock_news_sina（新浪财经）	获取单只股票的最新市场新闻、行业传闻、机构解读	跟踪市场舆情动向，及时发现可能影响股价的正面 / 负面新闻（如行业政策冲击、公司合作 / 违约消息）


## 沪深股市快讯	stock_market_news_cninfo（巨潮资讯）	沪深股市实时快讯，含个股突发消息	快速捕捉个股临时重大事件（如突发停复牌原因、监管问询回复等）



# 二、核心风险评估接口（直接关联个股风险点）
接口类别	接口标识	核心功能	风险评估作用
## ST 股票相关	stock_st_info	获取沪深 A 股 ST、*ST 股票列表及风险警示原因	直接锁定高风险标的，明确风险类型（如财务亏损、违规担保等）


## 停复牌信息	stock_suspension_info	查询个股停复牌日期、原因及复牌安排	停复牌往往隐含重大风险（如重大违规调查、经营危机），是风险预警核心接口


## 财务风险相关	stock_financial_risk_zjw（证监会财务风险预警）	获取上市公司财务风险等级、预警指标	量化评估财务风险（如流动性危机、盈利能力恶化等）


## 股权质押数据	stock_equity_pledge_detail	个股股权质押明细（质押比例、质押方、预警线）	高质押比例对应平仓风险，可跟踪质押变动判断股权稳定性


## 限售股解禁	stock_restricted_shares	个股限售股解禁日期、解禁数量、解禁股东	大规模解禁可能引发股价下行压力，属于短期流动性风险核心指标


## 股东增减持	stock_shareholder_change	大股东、董监高增减持明细（数量、价格、时间）	减持行为暗示内部对公司信心不足，是重要风险信号；增持则反之


## 龙虎榜风险信号	stock_dragon_tiger_list	龙虎榜买卖席位、成交金额、机构动向	识别个股短期炒作过度、游资集中撤离等风险，或机构持仓稳定性

# 三、辅助风险判断接口（间接印证风险趋势）
接口标识	核心功能	辅助价值
## stock_performance_forecast	上市公司业绩预告（预增 / 预减 / 预亏）	业绩大幅预亏直接反映经营风险，预告变脸暗示财务数据可信度风险


## stock_audit_opinion	上市公司财报审计意见	非标准审计意见（如保留意见、无法表示意见）暗示财务数据风险


## stock_margin_trading	融资融券交易明细（融资余额、融券余额）	融资余额过高→杠杆风险；融券余额激增→市场看空情绪浓厚


## stock_block_trade	大宗交易数据（成交价格、数量、买方 / 卖方）	低价大宗交易可能反映股东对公司未来预期悲观，隐含估值下调风险


## stock_industry_policy	行业政策动态	跟踪行业利空政策（如监管收紧、产能限制）对个股的间接冲击


补充说明
AKShare 的新闻类接口多依托第三方平台（东方财富、巨潮资讯等），数据实时性较强，可搭配 ts_code（股票代码）筛选单只个股的相关内容；
风险类接口覆盖 “财务风险、股权风险、市场情绪风险” 三大核心维度，可组合使用（如结合 stock_st_info+stock_equity_pledge_detail 判断高风险股）；
部分接口需指定参数（如日期、股票代码），调用时需按文档要求传入（示例：ak.stock_announcement(ts_code="600000.SH", start_date="20240101")）。

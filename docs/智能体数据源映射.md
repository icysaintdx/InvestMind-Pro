## 📝 前端显示格式

### 数据源对象格式
```javascript
{
  source: '数据源名称',
  count: 数据条数,
  description: '数据描述（可选）'
}
```

### 示例
```javascript
agentDataSources.value['fund_flow'] = [
  { source: '北向资金数据', count: 1, description: '沪深港通实时流向' },
  { source: '主力资金数据', count: 1, description: '大单成交监测' },
  { source: '融资融券数据', count: 1, description: '两融余额变化' },
  { source: 'AKShare', count: 3, description: '资金流向数据接口' }
]
```

## 🔧 常见问题

### Q1: 策略列表显示空白？

**检查**:
1. 后端服务器是否运行？
2. 前端服务器是否运行？
3. 浏览器控制台有无错误？

**解决**:
```bash
# 重启后端
Ctrl+C
python -m uvicorn backend.server:app --reload

# 刷新浏览器
F5
```

---

### Q2: 回测运行失败？

**检查**:
1. 股票代码是否正确？
2. 日期范围是否合理？
3. 是否选择了策略？

**调试**:
- 打开浏览器控制台（F12）
- 查看错误信息
- 检查API响应

---

### Q3: CORS错误？

**检查**: server.py的CORS配置

**配置**:
```python
app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)
```

## 🎯 使用建议

### 智能分析
1. **选择合适的股票**
   - A股：6位数字（如：600519）
   - 港股：5位数字（如：00700）
   - 美股：字母代码（如：AAPL）

2. **等待分析完成**
   - 通常需要1-2分钟
   - 21个智能体依次分析
   - 可以查看实时日志

3. **阅读分析报告**
   - 关注投资建议
   - 查看风险评估
   - 参考多空观点

---

### 策略回测
1. **选择合适的策略**
   - 技术分析：适合短期交易
   - 价值投资：适合长期持有
   - AI合成：综合多种因素

2. **设置合理参数**
   - 回测期间：至少3个月
   - 初始资金：建议10万以上
   - 策略参数：使用默认值或微调

3. **分析回测结果**
   - 查看收益曲线
   - 关注最大回撤
   - 评估夏普比率

## 📊 性能指标说明

### 回测指标

**收益指标**:
- 总收益率：整个回测期间的收益
- 年化收益率：折算成年化的收益
- 最大回撤：最大的资金回撤幅度

**风险指标**:
- 夏普比率：收益风险比（>1为好）
- 胜率：盈利交易占比
- 盈亏比：平均盈利/平均亏损

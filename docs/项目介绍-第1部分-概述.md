## 📌 项目概述

**InvestMindPro（阿尔法议会）** 是一个革命性的AI驱动投资决策系统，通过**21个专业AI智能体**的协同工作，模拟真实投资机构的决策流程，为投资者提供**全方位、多维度、深层次**的投资分析和决策支持。

```
传统投资分析 ❌                    InvestMindPro ✅

├─ 缺乏验证                        ├─ 闭环验证系统
└─ 难以追溯                        └─ 完整决策链记录
```

## 🎯 为什么选择 InvestMindPro？

### 传统投资分析的痛点

❌ **单一视角** - 只有一个分析师的观点，容易片面  
❌ **主观判断** - 缺乏量化依据，难以验证  
❌ **信息过载** - 海量新闻无法筛选，错过关键信息  
❌ **缺乏风控** - 只关注收益，忽视风险管理  
❌ **难以追溯** - 决策过程不透明，无法复盘  

### InvestMindPro 的解决方案

✅ **多维分析** - 21个智能体，8个维度，全方位覆盖  
✅ **量化决策** - 情绪评分、风险评级、目标价位  
✅ **智能过滤** - 360条新闻过滤为83条，信息密度+300%  
✅ **三方风控** - 激进/保守/中立，全面评估风险  
✅ **完整记录** - 从数据采集到最终决策，全程可追溯

## 🚀 快速开始

### 环境要求

- **Python**: 3.8+
- **Node.js**: 14+
- **操作系统**: Windows/Mac/Linux

### 一键启动（Windows）

```bash
# 1. 克隆项目
git clone https://github.com/yourusername/InvestMindPro.git
cd InvestMindPro

# 2. 配置API密钥
# 编辑 backend/.env 文件

# 3. 一键启动
START_ALL.bat
```

### 手动启动

```bash
# 1. 安装Python依赖
cd backend
pip install -r requirements.txt

# 2. 启动后端
python server.py

# 3. 安装前端依赖（新终端）
cd alpha-council-vue
npm install

# 4. 启动前端
npm run serve
```

### 访问系统

- 🌐 **前端界面**: http://localhost:8080
- 🔧 **后端API**: http://localhost:8000
- 📚 **API文档**: http://localhost:8000/docs

## 📖 使用指南

### 基础使用

1. **输入股票代码**
   - 支持A股6位代码（如：000001）
   - 自动识别沪深市场

2. **开始分析**
   - 点击"开始分析"按钮
   - 系统自动启动21个智能体

3. **实时查看**
   - 观察思维链展示
   - 查看数据源统计
   - 阅读分析结果

4. **查看报告**
   - 切换专业版/白话版
   - 导出MD/HTML/PDF
   - 保存历史记录

### 高级功能

#### 配置智能体模型

```
1. 点击智能体卡片右上角⚙️
2. 选择AI模型（Gemini/DeepSeek/Qwen/SiliconFlow）
3. 调整温度参数（0.0-1.0）
4. 保存配置
```

#### 配置白话解读员

```
1. 切换到报告的"白话版"
2. 点击⚙️配置按钮
3. 选择模型和温度
4. 保存配置
```

#### 自定义样式

```
1. 点击右上角"样式配置"
2. 选择主题（深色/浅色）
3. 调整字体大小
4. 保存设置
```

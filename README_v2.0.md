## ⚠️ 重要提示

### 回测局限性
1. 历史表现不代表未来收益
2. 回测结果依赖数据质量
3. 实盘交易有额外成本和滑点
4. 市场环境变化会影响策略表现

### 风险控制
1. ✅ 始终设置止损
2. ✅ 控制单笔仓位（建议≤30%）
3. ✅ 分散投资（使用多个策略）
4. ✅ 定期评估和调整
